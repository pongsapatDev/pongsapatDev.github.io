
-- คำสั่งสำหรับ GIT HUB --
git clone = ดึงข้อมูลจากเว๊ปลงโฟลเดอร์

git add . = เพื่อ save 

git mommit -m "delete all file for w701" = save ข้อมูลลง Local 

git push origin = เปลี่ยนสิ่งที่เกิดขึ้นไปยังเว็ปและในเครื่อง

git status = ดูสเตตัสโปลเดอร์

git push origin = save ลง web github 

git reset --hard(ใส่เลข commit)  = เพื่อย้อนกลับไปไฟล์ก่อนหน้านี้

git push origin -f = อับเดตและยืนยันการลบไฟล์นี้ออกไปจาก Local และ ใน เครื่อง
